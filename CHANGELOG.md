# Changelog

All important changes to QXP02V01 are recorded in this file.

## [V01] - Initial Release

### Added

- Webots e-puck simulation environment
- Differential-drive motor control
- Eight infrared proximity sensor integration
- Autonomous forward navigation
- Front obstacle detection
- Smooth steering correction
- Left and right path analysis
- Finite State Machine controller
- Left-turn and right-turn states
- Reverse recovery behaviour
- Trapped-condition detection
- Recovery-turn behaviour
- Alternating turn decisions
- Runtime navigation statistics
- Easy test world
- Medium test world
- Hard test world
- Recovery test world
- Project README
- Test-results documentation

### Known Limitations

- No mapping
- No localization
- No path planning
- No destination-based navigation
- No SLAM
- No computer vision
- No permanent environment memory
- Reactive navigation only
